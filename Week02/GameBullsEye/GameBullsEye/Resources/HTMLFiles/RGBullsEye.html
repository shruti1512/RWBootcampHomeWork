★ RGB Bull's Eye ★

This is the awesome game of a 3-slider variation of Classic BullsEye. The target is a random RGB color. The user drags the R, G, B sliders to try to match the target. As you move the sliders the color from RGB values apperas besides the target color block.

Your goal is to move the sliders so as to make the color as close as to the target color block. The closer you are, the more points you score.

Enjoy!
